# Muhammad Huzaifa Portfolio

A lightweight React + TypeScript single-page portfolio for Resume Writing and MS Office services.

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```
2. Run development server:
   ```bash
   npm run dev
   ```
3. Build for production:
   ```bash
   npm run build
   ```

## Deployment

This site is easy to deploy to Vercel, Netlify, GitHub Pages, or Azure Static Web Apps. Simply connect your repository and use the `npm run build` output.
